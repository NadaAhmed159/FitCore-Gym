# 💪 FitCore Gym Landing Page

A modern multi-section gym landing page built using **HTML5** and **CSS3** as part of my Front-End learning journey.

This project focuses on practicing core web development fundamentals such as layout structure, float layouts, and clean UI design.

---

## 🚀 Live Demo
🔗 https://nadaahmed159.github.io/FitCore-Gym/

## 📌 Features

- ✅ Clean and modern gym UI
- ✅ Well-structured multi-section layout
- ✅ Hero section with clear visual hierarchy
- ✅ "Why Choose Us" multi-column layout
- ✅ Testimonials section (social proof)
- ✅ Membership / Contact form
- ✅ External CSS for maintainability

---

## 🛠️ Tech Stack

- **HTML5** — semantic structure  
- **CSS3** — styling and layout  
- **Float Layouts** — element positioning
